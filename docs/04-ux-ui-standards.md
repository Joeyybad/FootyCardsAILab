# 04 - UI/UX Standards: Stadium Design Language

## 1. Aesthetics
- **Style**: Glassmorphic, sports-broadcast aesthetic.
- **Themes**: "Laboratory" (Dark) and "Stadium" (Light).
- **Typography**: Oswald (Headers), Inter (UI).

## 2. Core Interactions
- **Pack Reveal**: Zoom/fade animation with "Sparkles" reveal.
- **Battle Arena**: Dynamic comparison grid highlighting superior statistics in green.
- **Responsiveness**: Grid adapts column counts based on viewport width.
